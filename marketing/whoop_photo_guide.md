# WHOOP Photo Editing Guide — ສະໄຕລ໌ໄວໜຸ່ມ ເຂົ້າກັບທຸກເພດທຸກໄວ

ໃຊ້ Canva, Photoshop, ຫຼື CapCut ທຳຕາມຄຳແນະນຳລຸ່ມນີ້.

---

## ຮູບ 1 — WHOOP ສີດຳ (Black Band / One)

**ສະໄຕລ໌:** Energy / Street / Trendy

| ລາຍການ | ລາຍລະອຽດ |
|--------|-----------|
| Background | ສີດຳ (#0A0A0A) ຫຼື gradient ດຳ→ຟ້ານ້ຳເງິນເຂັ້ມ |
| Accent Color | Neon Cyan `#00F5FF` ຫຼື Lime Green `#AAFF00` |
| Text ຫຼັກ (ໃຫຍ່) | **"ດູແລສຸຂະພາບ"** ຕົວໃຫຍ່ ສີຂາວ ຢູ່ເທິງ |
| Text ຮອງ | **"ສ້າງນິໄສດີ ທຸກໆວັນ"** ສີ Neon ຢູ່ລຸ່ມ |
| Badge ສ່ວນຫຼຸດ | ວົງມົນ/starburst ສີ Neon: **"ສ່ວນຫຼຸດ 1,000฿"** |
| Font | Kanit Bold / Noto Sans Lao Bold |
| Effect | Light streak / glow ສີ Cyan ຜ່ານໂຕ band |
| Composition | ສິນຄ້າກາງຮູບ, text ເທິງ-ລຸ່ມ, ຂອບໄຟ Neon |

---

## ຮູບ 2 — WHOOP ສີຂາວ/ຂີ້ເທົ່າ (Gray Band / Peak & Life)

**ສະໄຕລ໌:** Clean Lifestyle / Health / Minimal

| ລາຍການ | ລາຍລະອຽດ |
|--------|-----------|
| Background | ສີຂາວ (#FAFAFA) ຫຼື gradient ຂີ້ເທົ່າອ່ອນ |
| Accent Color | Sage Green `#A8C5A0` ຫຼື Blush Pink `#F2A7A7` |
| Text ຫຼັກ (ໃຫຍ່) | **"ຮ່ອງລອຍສຸຂະພາບ"** ສີດຳ/ຂີ້ເທົ່າເຂັ້ມ |
| Text ຮອງ | **"WHOOP One | Peak | Life"** ສີ Accent |
| Badge ສ່ວນຫຼຸດ | ສີ່ຫຼ່ຽມມົນ ສີ Accent: **"ຮັບສ່ວນຫຼຸດສູງສຸດ 1,000฿"** |
| Font | Kanit Medium / Noto Sans Lao Regular |
| Effect | ເງົາອ່ອນ (soft shadow) ໃຕ້ product |
| Composition | ສິນຄ້າ 2/3 ລ່ຽງຂວາ, text ຊ້າຍ ຫຼື ສ່ວນລຸ່ມ |

## ໝາຍເຫດ

- ໃຊ້ຮູບສິນຄ້າທີ່ຕັດ background ອອກ (PNG ໂປ່ງໃສ) ຈະດູດີກວ່າ
- ຢ່າໃສ່ text ຫຼາຍເກີນ — ໂຈະຕ່ຳ 3–4 ເສັ້ນຕໍ່ຮູບ
- Export ດ້ວຍ JPEG 90%+ ຫຼື PNG ເພື່ອ quality ດີສຸດ
